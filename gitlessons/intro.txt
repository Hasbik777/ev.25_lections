GIT - распределенная система контроля версий

Это система для отслеживания и ведение истории изменения файлов, в вашей проекте.

Репозитор - хранилище вашего кода и истории его изменений

GitHub - Веб сайт для размещения git-респозиториев в совместной разработке проектов

Команда Git:
1. git init - она создает новый локальный гит репозиторий у вас на пк, в той директории в которой была прописана команда. (Команда вводиться один раз в начале)
2. git add - команда для добавления файлов в систему отслеживания гита, после этой команды изменения начинают отслеживаться гитом.
git commit -m '<comment>" - как только достигаем определенного момента в разработке, то тогда мы сохраняем и комментируем изменения в локальной репозиторий
